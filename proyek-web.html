Web Informasi Bengkel

Situs bengkel motor & mobil untuk menampilkan layanan, galeri, kontak, dan fitur booking online dengan konfirmasi melalui WhatsApp.

Tujuan: Menyediakan website informasi bengkel motor dan mobil yang mudah digunakan, menampilkan layanan, galeri, kontak, dan mendukung booking online dengan konfirmasi melalui WhatsApp.

Fitur: hero, informasi layanan, galeri, kontak, form booking online, dan integrasi chat WhatsApp.

Ringkasan Proyek

Ini adalah proyek website informasi bengkel yang dibuat untuk memudahkan pelanggan menemukan layanan servis mobil dan motor, melihat galeri hasil pekerjaan, menghubungi lewat WhatsApp, dan membuat booking layanan secara online.

Halaman Utama

index.html — hero, fitur singkat, galeri, footer, dan skrip client untuk menyiapkan pengiriman booking serta membuat tautan WhatsApp.

booking.html — formulir cepat (nama, WA, jenis kendaraan, layanan, catatan) yang dikirim via fetch ke API lokal.

layanan.html — daftar layanan bengkel seperti servis rutin, perbaikan mesin, rem/suspensi, dan diagnosa elektronik.

kontak.html — alamat, nomor WA, jam operasional, dan tombol chat WA.

Backend & Penyimpanan

Server Express dijalankan pada http://localhost:3000 dengan endpoint:

GET /api/bookings — ambil daftar booking.

POST /api/bookings — simpan data booking ke bookings.json.

Data booking disimpan ke file bookings.json (saat ini berisi []).

Alur Booking

Pengguna mengisi form di booking.html.

Form dikirim via POST /api/bookings ke server lokal.

Server menyimpan data ke bookings.json.

Halaman membuat tautan https://wa.me/... untuk melanjutkan chat WhatsApp.

Catatan Teknis Singkat

Skrip client di index.html mengakses elemen form/btn dan hanya bekerja jika halaman itu dimuat atau server Node dijalankan agar request POST berhasil.
