Web Portofolio Pribadi

Web portofolio ini dibuat dengan HTML, CSS, dan JavaScript untuk menampilkan proyek dan kemampuan desain web.

Deskripsi Proyek

Situs portofolio ini dirancang sebagai proyek web yang responsif, mudah dinavigasi, dan cocok untuk presentasi personal atau bisnis kecil.

Web Bengkel — halaman proyek dengan deskripsi layanan bengkel, fitur booking, dan galeri.

SERSHOP — tampilan marketplace bertema futuristik untuk produk digital.

Peran Saya

Mendesain tata letak dan struktur halaman agar tampil profesional dan mudah dibaca.

Mengembangkan antarmuka menggunakan HTML, CSS, dan JavaScript untuk interaksi yang lancar.

Menghubungkan semua halaman agar navigasi menjadi jelas dan pengalaman pengunjung lebih baik.

Menerapkan gaya responsif agar situs terlihat baik di desktop maupun perangkat mobile.

Manfaat untuk Klien

Meningkatkan kredibilitas online dengan situs portofolio profesional.

Membantu menjelaskan layanan dan kemampuan secara jelas kepada calon klien.

Menyediakan akses kontak yang mudah untuk permintaan kerja atau kolaborasi.

Membuat presentasi proyek lebih menarik dan mudah dibagikan.
