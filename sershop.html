Penjelasan Website: SERSHOP MARKETPLACE

Website ini adalah platform marketplace e-commerce futuristik dengan desain cyberpunk.

🎯 Tujuan Utama

Platform belanja online modern yang menghubungkan pembeli (User), penjual (Seller), dan admin dalam satu ekosistem.

Sistem Multi-Role

  • User (Pembeli): browsing & mencari produk, menambah ke wishlist dan shopping cart, checkout dan pembayaran, tracking pesanan, memberikan review & rating.
  • Seller (Penjual): mengelola katalog produk, menerima & memproses pesanan, melihat statistik penjualan, chat dengan pembeli.
  • Admin (Administrator): monitor semua user dan seller, mengelola kategori produk, melihat laporan transaksi, moderasi konten.

Teknologi Stack

  • Backend: PHP Native (tanpa framework)
  • Database: MySQL
  • Frontend: HTML5, CSS3, Vanilla JavaScript
  • Server: Apache/XAMPP

Struktur Utama

  • pages: halaman auth, user, seller, admin, public
  • api: API endpoints untuk cart, wishlist
  • config: konfigurasi database & session
  • components: reusable navbar, footer, head
  • css & js: styling & interaksi
  • uploads: penyimpanan gambar produk & profil

Singkatnya: ini adalah web belanja online lengkap dengan fitur marketplace, sistem multi-peran, dan desain modern cyberpunk yang siap produksi! 🚀
